With this modification reporting tool I was able to visualize a 2D map into a 3D rotating visualization of a World-map. The moment you hover over a country, the verge were coloured highlithed. If you want to know more about the hovered country you should click on it. The report is a simple d3.js bubble chart with random data. This tool is a great visualization for companies which covering the global market. I would love to use this visualization for companies which wants to offer their costumer some awesome features based on their data. This project conntains http-methods for generating the choosen country to the depending data. Country data is represented as polygons with model-data of the worlds countries. All features are wrapped into a flask web-application to use this on your own web-server.
